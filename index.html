Questionario

  1. Escreva a alternativa correta para "Na tag < input > com atributo type="text", para que serve o atributo "placeholder"?"

    2. R: placeholder funciona como um texto de apresentação sobre o que é esperado para o campo em questão.


  2. Na tag < input > com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?

    3. R: A pseudo-classe :checked de um seletor CSS representa um elemento radio (< input type="radio" >), checkbox (< input type="checkbox" >) ou option (< option > em um < select >) que está marcada ou alternado para um estado ligado.


  3. Na tag < input > para que serve o atributo "value"?

    4. R: Você deve usar o atributo value para definir o valor enviado por este item.


  4. Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

    5. R: O atributo necessario para isso é "type:"radio""


  5. Para que serve a tag < label >? Para que serve o atributo "for" nessa tag?

    6. R:A teg < label >especifica qual o "rótulo" do inpute, e ajuda na experiência do usuário durante a utilização e preenchimento do formulário. É o atributo de entrada para qual o < label > se destina. Seu valor deve ser igual ao valor do atributo id do elemento controle de entrada (< input >).


  6. Na tag < option > para que serve o atributo "selected"?

    7. R: Pode ser utilizado de diferentes maneiras em uma página, como para construir um menu de navegação, exibir listas para a seleção de um ou vários itens, ser adicionado em um formulário, etc. Também é possível configurá-lo de diferentes formas, como fazer com que a seleção de um elemento seja obrigatória, ativar ou desativar os elementos exibidos na lista, entre outros.


  7. Nas tags de formulário para que serve o atributo "required"?

    8. R: O atributo "required" em uma tag de formulário é usado para especificar que um campo é obrigatório e deve ser preenchido antes que o formulário possa ser enviado.


  8. Para que serve a tag < textarea >?

    9. R: A tag < textarea > é usada para criar uma área de texto de várias linhas em um formulário, onde o usuário pode inserir uma quantidade maior de texto.


  9. Na tag < form > para que serve o atributo "action"?

    10. R: O atributo "action" em uma tag < form > é usado para especificar o URL para onde os dados do formulário serão enviados após o envio do formulário.


  10. Na tag < form > para que serve o atributo "method"?

    11. R: O atributo "method" em uma tag < form > é usado para especificar o método HTTP que será usado para enviar os dados do formulário para o servidor. Os dois métodos mais comuns são "GET" e "POST".


  11. Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o
    valor do atributo "method" na tag < form >?

    12. R: Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, o valor do atributo "method" não tem nenhuma relação com isso, é necessário usar o tipo de entrada "password" na tag < input > correspondente à senha.


  12. Qual valor deve ser informado no atributo "method" na tag < form > em um formulário que será usado para fazer upload de arquivos?

    13. R: Para fazer upload de arquivos, o valor do atributo "method" deve ser definido como "POST".


  13. O que as tags < fieldset > e < legend > produzem?

    14. R: As tags < fieldset > e < legend > são usadas para agrupar e identificar elementos de formulário relacionados. A tag < fieldset > é usada para agrupar um conjunto de elementos de formulário em um bloco, enquanto a tag < legend > é usada para fornecer uma descrição ou legenda para o grupo de elementos de formulário dentro do < fieldset >.
